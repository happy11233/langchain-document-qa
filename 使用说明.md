# 智能文档问答系统 - 使用说明

## 功能介绍

这是一个基于 LangChain 和阿里云通义千问的智能文档问答系统，可以：
- 上传 PDF、Word、TXT、Markdown 文档
- 用自然语言提问
- AI 基于文档内容回答问题
- 显示答案来源

## 使用步骤

### 1. 获取 API Key

访问阿里云通义千问官网：https://dashscope.aliyun.com/

1. 注册/登录阿里云账号
2. 进入控制台
3. 创建 API Key
4. 复制你的 API Key（格式：sk-xxxxxx）

### 2. 启动系统

```bash
# 进入项目文件夹
cd "Q&A system based on LangChain"

# 激活虚拟环境
source .venv/bin/activate

# 运行
streamlit run app.py
```

### 3. 配置系统

1. 在左侧边栏找到 "⚙️ 系统配置"
2. 输入你的 API Key
3. 点击 "🔄 登录系统" 按钮
4. 等待系统登录完成

### 4. 上传文档

1. 在左侧边栏找到 "📤 上传文档"
2. 点击 "Browse files" 选择文件
3. 支持的格式：PDF、Word（.docx）、TXT、Markdown（.md）
4. 可以一次选择多个文件
5. 点击 "🚀 开始处理" 按钮
6. 等待文档处理完成

### 5. 开始提问

1. 在主界面底部的输入框输入问题
2. 按回车发送
3. AI 会基于上传的文档内容回答
4. 点击 "📄 查看来源" 可以看到答案的出处

## 功能说明

### 文档管理
- **文档段落**：显示知识库中的文档片段数量
- **文件数量**：显示已上传的文件数量
- **已上传文件**：列出所有已上传的文件名

### 操作按钮
- **🗑️ 清空对话**：清除聊天记录，但保留知识库
- **💣 清空知识库**：删除所有上传的文档和聊天记录
- **🚪 退出登录**：退出当前账号，清空所有数据（需要确认）

## 切换 API Key

如果需要使用其他 API Key：

1. 点击左侧边栏的 "🚪 退出登录" 按钮
2. 系统会提示：退出后将清空所有数据
3. 点击 "✅ 确认退出"
4. 输入新的 API Key
5. 点击 "🔄 登录系统"

**注意**：退出登录会清空：
- 聊天记录
- 上传的文档
- 知识库数据

## 注意事项

1. **API Key 安全**：
   - 不要分享你的 API Key
   - 系统使用密码框输入，不会显示明文
   - API Key 只保存在当前会话，关闭浏览器后需要重新输入

2. **文档格式**：
   - PDF：支持文字版 PDF（扫描版需要 OCR）
   - Word：只支持 .docx 格式（不支持 .doc）
   - TXT：需要 UTF-8 编码
   - Markdown：支持标准 Markdown 格式

3. **文档大小**：
   - 建议单个文件不超过 10MB
   - 文档会被切分成小块（默认 1000 字符/块）

4. **费用**：
   - 使用阿里云通义千问 API 会产生费用
   - 具体费用请查看阿里云官网定价

## 常见问题

### Q: 初始化失败怎么办？
A: 检查 API Key 是否正确，确保网络连接正常

### Q: 上传文档失败？
A: 
- 检查文件格式是否支持
- 确认文件没有损坏
- 查看错误提示信息

### Q: AI 回答不准确？
A: 
- 确保上传的文档包含相关信息
- 尝试换个方式提问
- 文档内容越详细，回答越准确

### Q: 如何更换 API Key？
A: 
1. 点击左侧 "🚪 退出登录" 按钮
2. 确认退出（会清空所有数据）
3. 输入新的 API Key
4. 点击 "🔄 登录系统"

### Q: 退出登录会丢失数据吗？
A: 
是的，退出登录会清空：
- 所有聊天记录
- 上传的文档
- 知识库数据

如果需要保留数据，请不要退出登录。
